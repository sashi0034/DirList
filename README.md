# DirList
簡易エクスプローラーのようなもの

標準エクスプローラーのピン止め出来る個数を増やしたかったのです。

- 青い❌ボタンで削除

- 緑の🗒️ボタンでクリップボードにコピー

- フォルダパスが書いてあるボタンをダブルクリックすると、VSCodeかエクスプローラーでフォルダを開きます。
